# django-website-with-cms

This is a simple website with cms panel which I wrote with Django. This page has such features as:

- displaying all articles with main picture
- displaying single article with main picture, publication date, author name and all comments
- adding comment to articles
- if you have user account, you can login and add new article with main picture, display your articles at dashboard and edit ones.
